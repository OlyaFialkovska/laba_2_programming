Block1:
Create files f1, f2, f3, f4. Formulas for formation
files f2, f3, f4 are taken from other options. Organize an exchange
components between files in accordance with the scheme: f1 => f3,
f2 => f4, f3 => f2, f4 => f1. It is allowed to use only
one auxiliary file.

Block2:
Create a character file. Move to the end of the file
its first component, which is a vowel letter and stands between
with two consonants.

Block3:
Given text files f and g. Write the text to file h first
file f and then the text of file g, skipping all blank lines.
